# Excel Dashboard Project

## Overview
This project demonstrates an interactive Excel dashboard designed to analyze and visualize data in a clear and meaningful way. The dashboard helps identify key trends and insights that support data-driven decision making.

## Objective
To showcase my Excel and data analysis skills by creating a professional dashboard suitable for a Data Analyst fresher role.

## Project Description
- Raw data was cleaned and organized using Excel  
- Key metrics were calculated using Excel formulas  
- Interactive charts, KPIs, and slicers were created  
- The dashboard allows users to filter data dynamically and understand performance quickly  

## Key Features
- Interactive Excel dashboard  
- KPIs for quick performance tracking  
- Charts for visual insights  
- Slicers for dynamic filtering  
- Clean and user-friendly layout  

## Tools & Technologies
- Microsoft Excel  

## Insights
- Identified trends and patterns from the data  
- Simplified complex data into visual format  
- Improved understanding of overall performance
